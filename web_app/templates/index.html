Kết nối API Meta tự động

Chế độ tự động

Hệ thống sẽ chạy đồng bộ Meta API ngay với link sheet hiện tại, tự ghi dữ liệu chi phí nếu token và quyền đã sẵn sàng.

Nếu báo thiếu quyền hoặc token hết hạn, dùng checklist bên phải để nhân viên kết nối đúng chuẩn.

Checklist hướng dẫn nhân viên

  1. Vào Business Settings của tài khoản quảng cáo cần kết nối.
  2. Cấp quyền ads_read (và ads_management nếu cần) cho app/token đang dùng.
  3. Đảm bảo token chưa hết hạn, có quyền trên đúng ad account.
  4. Dán đúng link sheet, sau đó bấm Đồng bộ API tự động ngay.
  5. Nếu vẫn lỗi, gửi lại nội dung lỗi để admin cập nhật token quyền cao hơn.
